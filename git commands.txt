LISTA DE COMANDOS GIT HUB

//Status inicial do git e arquivos que estao pendentes para enviar ao repositorio.

	1- git init
	2- git status
	
//Adicionar os arquivos que serao mandados ao repositorio.

	3- git add [arquivo]
	3- git add .
	3- git add *txt
	


//Sempre adicione uma descricao sobre o repositorio, isso serve pra organizar e manter atualizado a funcao da pasta.

	4- git commit -m "descricao do repositorio" 	
	5- git log

//Provavelmente apos esse passo o gir precisara ser configurado, se esse for seu primeiro acesso. Basta seguir as instrucoes.


//Antes de adicionar o arquivo é preciso informar o repositorio para onde os arquivos serao enviados.
Crie o repositorio no Github, e copie o link URL.

	6- git remote add origin [endereco url]

//No primeiro push é preciso definir qual o branch será enviado o arquivo. Inicialmente sempre manda para master. 

	7- git push --set-upstream origin master

	8- git pull origin master
	9- git diff head
	10- git diff --staged
	11- git reset [pasta/arquivo]
	12- git checkout -- [arquivo]
	13- git branch [novo branch (clean_up)]
	14- git checkout [(clean_up)]
	15- git rm [arquivo]
	16- git chekout master
	17- git merge clean_up
	18- git branch -d [branch(clean_up)]
	19- git push
